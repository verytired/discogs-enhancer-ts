# Changelog

## v0.2.0 (2026-02-08)
### Added
- **Marketplace Infinite Scroll**: Automatically loads the next page of items in the marketplace listing.
- **Block Sellers**: Adds a "Block" button next to sellers in the marketplace to hide their listings. Manage blocked sellers in Options.
- **Sort by Total Price**: Adds a button to sort marketplace items by total price (Item Price + Shipping).
- **Actual Dates**: Displays absolute dates (YYYY-MM-DD) alongside relative dates (e.g., "2 months ago").
- **Modern Options UI**: Redesigned options page with improved layout, toggle switches, and dark mode support.
- **Context Menu Search**: Right-click on selected text to search on Discogs.

### Fixed
- **Dark Mode Visibility**: Fixed headers being invisible (white text on white background) in dark mode.
- **Options Page Layout**: Fixed layout issues when opening options in a full tab.
