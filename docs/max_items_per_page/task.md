# タスク内容

Discogsのアーティストページやレーベルページを閲覧する際、リリース一覧の表示数（`limit`パラメータ）が常に最大値である500になるように機能を追加する。

## 目標
1. `https://www.discogs.com/artist/*` および `https://www.discogs.com/label/*` パスにアクセスした際に、URLのクエリパラメータに `limit=500` がない場合は付与してリロードするようにする。
2. 他の拡張機能の設定と同様に、オン/オフを切り替えられるスイッチをオプション画面の「General」セクションに追加する。デフォルト設定はオンにする。
